Español.
En caso de que posiblemente tuvieras un error con el compilador y este deja de funcionar, deberas cambiar la compatibilidad de pawncc.exe a Windows 7, esto es solo si llegase a dejar de compilar o no compila algun filterscript. (Es un pequeño bug del compilador)
Esta Gamemode usa los Includes default de SAMP, no usa includes modificados por terceros, eres libre de actualizar el compilador y los includes.

NOTAS:
En caso de que al compilar tengas los siguientes problemas:
1. '-DD:\GamemodeBase-MySql-master\Version.p(0) : fatal error 100: cannot read from file: "'-DD:\GamemodeBase-MySql-master\Version.p"
2. '-DD:\gamemode.p(0) : fatal error 100: cannot read from file: "'-DD:\gamemode.p"

La recomendacion es mover todo lo que esta dentro de Version 1.0/Version 1.1 a una carpeta con un nombre distinto, en caso de que el nombre de la carpeta lleve espacios como por ejemplo "gamemode mi nombre", tendrias que cambiar el nombre a "gamemode-mi-nombre" o bien puedes usar "_" sino deseas usar "-".

Posdata: El que no respete los creditos, se la come doblada.


English.
In case you possibly have a compiler error and it stops working, you should change the compatibility of pawncc.exe to Windows 7, this is only if it stops compiling or does not compile any filterscript. (It's a small compiler bug)
This Gamemode uses the default includes from SAMP, it does not use includes modified by third parties, you are free to update the compiler and the includes.

GRADES:
In case when compiling you have the following problems:
1. '-DD:\GamemodeBase-MySql-master\Version.p(0) : fatal error 100: cannot read from file: "'-DD:\GamemodeBase-MySql-master\Version.p"
2. '-DD:\gamemode.p(0) : fatal error 100: cannot read from file: "'-DD:\gamemode.p"

The recommendation is to move everything that is inside Version 1.0/Version 1.1 to a folder with a different name, in case the folder name has spaces such as "gamemode my name", you would have to change the name to " gamemode-my-name" or you can use "_" if you don't want to use "-".

Postscript: Whoever does not respect the credits, eats it folded.
